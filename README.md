## Implementation

```
buildscript {
    repositories {
        maven { setUrl("https://jitpack.io") }
    }
}
```

Then add all or only needed library module dependencies to your module level build gradle:

```
dependencies {
    implementation 'com.github.ShiftHackZ.AndroidCoreMVI:mvi:<VERSION>'
}
```
